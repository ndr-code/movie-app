# Movie Explorer App - React & TMDB Integration

A modern, fully responsive movie exploration application built with React and TypeScript, integrated with The Movie Database (TMDB) API.

**Deployed at:** https://movie-app-for-you.vercel.app/

![Movie App](https://media.githubusercontent.com/media/triaji-dev/portofolio-assets/refs/heads/main/movie-app/gif/preview-movie-app-for-you.gif)

---

## Overview

### Title
Modern Movie Explorer Application with Real-Time TMDB Integration

### Key Features
- **Dynamic Hero:** Auto-rotating slider showcasing trending movies
- **Browsing & Infinite Scroll:** Explore trending and new releases with seamless infinite loading
- **Fast Search:** Real-time search functionality with debouncing for performance
- **Comprehensive Details:** Full movie info, cast, genres, ratings, and integrated YouTube Trailer playback
- **Favorites System:** Save favorite movies using local storage persistence
- **Full Responsiveness:** Optimized for all devices (mobile, tablet, desktop)

---

## Core Technologies

- **Frontend:** React 18, TypeScript, Vite
- **Styling:** Tailwind CSS v4
- **API:** TMDB API, Axios
- **Advanced Features:** Custom Hooks, Infinite Scroll (Intersection Observer), Code Splitting
